Introduction

CSS stands for Cascading Style Sheets and is used to style HTML documents.

It controls colors, fonts, spacing, layouts, and the overall appearance of web pages.

CSS can change the appearance of a web page.

body { background-color: lightblue; }

Selectors

CSS selectors are used to target HTML elements that need styling.

Common selectors include element, class, and ID selectors.

p { color: red; }

Common Selectors are:

  • Element selector (p)
  • Class selector (.class-name)
  • ID selector (#id-name)
Colors

CSS allows developers to apply colors to text, backgrounds, and borders.

Colors can be specified using names, hexadecimal values, RGB, or HSL.

h1 { color: #3498db; }

Color formats:

  • Color names
  • Hex codes
  • RGB values
  • HSL values
Box Modelling

The CSS box model describes how elements are displayed as rectangular boxes.

It consists of content, padding, border, and margin.

div { margin: 10px; }

The box model consists of:

  • Content
  • Padding
  • Border
  • Margin
Flexbox

Flexbox is a one-dimensional layout system for arranging items in rows or columns.

It makes it easier to align and distribute space between elements.

.container { display: flex; }
